To-do List

JavaScript, Webpack HTML5, CSS3

Source Demo

This to-do app was create to help users to organize their tasks and make it easier. the rainbow colors were used to inspire joy and energy on users. With this app users can create maximum of 7 projects, create todos inside projects, set a day and time to each todo and set low, normal and high priority to each todo. It is possible to edit and delete todos, and mark todos as completed. To help them to organize, UniList let users see completed, incompleted and expired todos, see the next todo and also see todos ordered by date or by priority.
